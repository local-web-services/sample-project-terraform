# Contributing

Thank you for your interest in contributing to this project!

## Before You Start

Please raise a GitHub issue with your idea before starting any work. This helps to:

- Avoid duplication of effort
- Get early feedback on your proposed change
- Ensure the change aligns with the project's direction

## How to Contribute

1. **Open an issue** describing the change you'd like to make
2. Wait for feedback and approval from the maintainers
3. Fork the repository
4. Create a branch for your change
5. Make your changes
6. Submit a pull request referencing the issue
